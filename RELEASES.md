# Releases

## 1.0.0 (2026-06-28)

First stable release.

**Features:**
- 🎵 Music card with full-bleed album art, artist overlay, Apple Music / Spotify support
- 📅 Calendar with week strip, lunar date, English + Chinese month display
- ☀️ Weather via Open-Meteo API with CoreLocation
- 📋 Clipboard history with search, keyboard navigation, text + image support
- 📝 Editable snippets with keyboard navigation, inline editing
- ⌨️ Global hotkey `⇧⌘O` to toggle panel
- 🎨 Light / Dark theme with persistent preference
- ⎈ Menu bar icon with context menu (Show/Hide, Quit)
- macOS 14.0+ deployment target
